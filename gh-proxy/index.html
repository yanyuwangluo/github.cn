公告:

本站限制单日请求量超过10W自动停止。

如果是网页使用，直接将原链接粘贴到输入框点击下载即可加速。

如果使用其他工具进行文件下载，则在原链接前新增加速前缀：

Git 项目加速：git clone https://www.yanyu.ltd/< https 开头的项目地址>

Wget 加速下载：wget https://www.yanyu.ltd/<原来的文件链接>

google 或 jcenter 仓库镜像：https://www.yanyu.ltd/cfworker/<仓库地址>

加速任意链接：https://www.yanyu.ltd/<任意 http 或 https 链接>

在后面拼接的URL必须是 http:// 或者 https:// 开头

合格的URL 1: https://github.com/zwc456baby/file-proxy.git

合格的URL 2: https://github.com/zwc456baby/file-proxy/archive/master.zip

不合格的示例: github.com/zwc456baby/file-proxy/archive/master.zip

不合格的示例: www.baidu.com
